# Welcome to Termos

**Termos** is a powerful Go library for creating interactive terminal user interfaces (TUI). Built on top of Bubble Tea, it provides high-level components for quickly building console applications.

## Quick Start

Install the library and start creating TUI applications in minutes:

```go
package main

import (
    "log"
    "github.com/qzeleza/termos"
)

func main() {
    // Create a task queue
    queue := termos.NewQueue("My TUI Application")

    // Add confirmation task
    confirm := termos.NewYesNoTask("Confirmation", "Continue?")

    // Add selection task
    options := []string{"development", "staging", "production"}
    env := termos.NewSingleSelectTask("Select environment", options)

    // Add tasks to queue and run
    queue.AddTasks(confirm, env)
    if err := queue.Run(); err != nil {
        log.Fatal(err)
    }
}
```

## Optimized for Embedded Systems

Termos automatically detects and optimizes for embedded systems:

- **Memory Management** - load monitoring and string interning
- **Adaptive Color Schemes** - simplified colors for limited terminals
- **Auto-configuration** - router and IoT device detection with parameter tuning
