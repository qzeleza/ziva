# Color Customization

Ziva provides a rich color system for creating visually appealing TUI applications.

## Available Colors

```go
// Bright colors
ziva.GreenBright
ziva.RedBright
ziva.BlueBright
ziva.YellowBright
ziva.OrangeBright
ziva.CyanBright
ziva.MagentaBright
ziva.WhiteBright

// Dark colors
ziva.RedDark
ziva.YellowDark
ziva.OrangeDark
ziva.BlueDark
ziva.CyanDark
ziva.GrayDark
ziva.DarkGreen

// Special colors
ziva.LightBlue
ziva.GrayBright
ziva.Black
```

## Queue Color Configuration

```go
queue := ziva.NewQueue("Colorful App")

// Application name styling
queue.WithAppNameColor(ziva.BlueBright, true)  // blue and bold

// Title styling
queue.WithTitleColor(ziva.GreenBright, false) // green, normal weight

// Error colors
queue.SetErrorColor(query.ErrorColorRed)        // red errors
```

## Terminal Compatibility

Ziva automatically adapts colors based on terminal capabilities:

- **True color terminals** - full color palette
- **256-color terminals** - optimized color mapping
- **16-color terminals** - basic color fallback
- **Monochrome terminals** - styling with intensity only
