# История версий

<Info>
Ниже перечислены заметные изменения основных релизов Termos. Даты указаны в часовом поясе разработчика.
</Info>

## v1.2.4 · 22 сентября 2025
- Исправлена поломка `WithTimeout` в задачах подтверждения и выборов.
- Добавлены языко-независимые методы таймаута (`WithTimeoutYes/No`, комбинированные варианты).
- Обновлена документация API дополнительными функциями настройки.

## v1.2.3 · 21 сентября 2025
- Расширена `WithViewport` для списковых задач: добавлен параметр управления счётчиками элементов.

## v1.2.2 · 20 сентября 2025
- Добавлена обработка клавиш «влево/вправо» для ускоренной навигации в списках.

## v1.2.1 · 20 сентября 2025
- Реализована встроенная локализация служебных надписей (ru, en, tr, be, uk).

## v1.2.0 · 27 августа 2025
- Появилась функция `WithViewport`, ограничивающая высоту списков.

## v1.1.12 · 25 августа 2025
- `formatErrorEveryLine` получил аргумент `delNewLines`, что позволяет управлять переносами в сообщениях.

## v1.1.11 · 25 августа 2025
- Ошибки больше не обрезаются по фиксированной ширине — выводится полный текст.

## v1.1.10 · 24 августа 2025
- Переработан вывод ошибок и вынесены константы в модуль `defaults`.

## v1.1.0 · 22 августа 2025
- Таймауты стали доступны для всех типов задач с возможностью указать значение по умолчанию.

## v1.0.2 · 10 августа 2025
- Исправлены префиксы состояний в TUI.

## v1.0.0 · 8 августа 2025
- Первый стабильный релиз:
  - добавлены очередь задач, подсистема валидации и все базовые типы задач;
  - реализованы стили UI, набор утилит и система ошибок;
  - подготовлены демонстрационные приложения, документация и примеры для embedded-конфигураций;
  - покрытие тестами очереди, валидаторов, UI и утилит.

<Note>
История minor- и patch-релизов отображает ключевые изменения. Для подробных diff используйте `git log` или GitHub Releases.
</Note>
